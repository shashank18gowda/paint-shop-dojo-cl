# `@repo/eslint-config`

Shared ESLint configurations for the Paint Shop Dojo monorepo.

Exposes three flat-config presets:

- `@repo/eslint-config/base` — base JS/TS rules (used by non-React packages, e.g. `game-app`)
- `@repo/eslint-config/next-js` — Next.js apps
- `@repo/eslint-config/react-internal` — internal React libraries (e.g. `@repo/ui`)
